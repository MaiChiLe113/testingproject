Note: Shortest path program SP(G, a, b) which returns a path from node a (start_node) to node b (end_node) in graph G (Lecture 5, Canvas). |SP(G,a,b)| denotes the length of SP(G, a, b) (Lecture 5, Canvas).
MR1: Graph Transposition
Description: Reverse all directed edges, keep the same weights of the edges. With the constraint that all weights are non-negative and the program uses directed graph, the path's length should stay the same. MR1 checks for consistency between source and follow-up test case.
MR Property:			|SP(G, a, b)| = |SP(GT, b, a)|    (GT is the transposed graph)
